# Zig agent SDK example (SDK-6)

Consumes the C ABI from `feanorfs-ffi`. Build the shared library first (Zig links `libfeanorfs_ffi.so` / `.dylib`, not the static `.a`):

```bash
cargo build -p feanorfs-ffi
cargo build -p feanorfs-client   # feanorfs CLI for workspace setup

# Configure a local-hub workspace (needs at least one synced file for spawn snapshot)
mkdir -p /tmp/zig-ws && cd /tmp/zig-ws
/path/to/feanorfs start --local --workspace zig-demo --no-watch
echo seed > seed.txt
/path/to/feanorfs sync --no-watch

export FEANORFS_WORKSPACE=/tmp/zig-ws
cd examples/zig-agent && zig build
./zig-out/bin/zig-agent-demo
```

Requires **Zig 0.16+**. Header: `feanorfs-ffi/feanorfs.h` (generated by `cbindgen` on build).

See [docs/agent-api.md](../../docs/agent-api.md) for JSON shapes returned by each FFI call.
